class quaterion.train.encoders.cache_config.CacheConfig(cache_type: typing.Optional[quaterion.train.encoders.cache_config.CacheType] = None, mapping: typing.Dict[str, quaterion.train.encoders.cache_config.CacheType] = <factory>, key_extractors: typing.Dict[str, typing.Callable[[typing.Any], typing.Hashable]] = <factory>, batch_size: typing.Optional[int] = 32, num_workers: typing.Optional[int] = None)¶

Bases: object

Class to be used in configure_cache of TrainableModel.

cache_type: cache type for single encoder which has no custom name set. In all other

cases use mapping.

mapping: mapping of encoder_name to CacheType, only specified inhere encoders

will be cached, the others will stay untouched.

key_extractors: mapping of encoders to key extractor functions required to cache

non-hashable objects.

batch_size: batch size to be used in CacheDataLoader during caching process. It does

not affect others training stages.

num_workers: num_workers to be used in CacheDataLoader during caching process. It

does not affect others training stages.

class quaterion.train.encoders.cache_encoder.CacheEncoder(encoder: quaterion_models.encoders.encoder.Encoder, key_extractor: Optional[Callable[[Any], Hashable]] = None)¶

Bases: quaterion_models.encoders.encoder.Encoder

Wrapper for encoders to avoid repeated calculations.

Encoder results can be calculated one time and reused after in situations when encoder’s layers are frozen and provide deterministic embeddings for input data.

Parameters

  • encoder – Encoder object to be wrapped.

  • key_extractor – function required to extract hashable key from complicated objects which can’t be hashed with default key extractor.
